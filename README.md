# JavaVerse

## Project Overview

JavaVerse is a Battle Royale inspired Java Learning Platform where players learn Java programming through missions, battles, coding challenges, rewards, and interactive gameplay.

## Vision

To make Java programming fun, interactive, and engaging through game-based learning.

## Technology Stack

- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- 3D Engine: Three.js + React Three Fiber
- Authentication: Spring Security + JWT
- AI: OpenAI API (Future)
